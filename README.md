# 🏡 WanderLust – Full Stack Airbnb Clone

A production-level full stack Airbnb-inspired web application where users can explore, create, edit, review, and manage property listings.  
Built using the MERN stack with authentication, cloud image uploads, interactive maps, and responsive UI.

---

# 🚀 Live Features

- 🔐 User Authentication & Authorization
- 🏠 Create, Edit & Delete Property Listings
- 📸 Cloudinary Image Upload Integration
- 🗺️ Interactive Maps with Mapbox
- ⭐ Review & Rating System
- 🔎 Search & Explore Listings
- 📱 Fully Responsive Design
- ☁️ MongoDB Atlas Cloud Database
- ⚡ RESTful APIs
- 🛡️ Secure Session & Flash Messages
- 🌐 Production Deployment Ready
